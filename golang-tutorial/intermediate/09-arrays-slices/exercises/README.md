# 第九章練習題

## 練習 1：動態數組實現

創建一個完整的動態數組類：

要求：
- 實現動態擴容和縮容
- 支援插入、刪除、查找操作
- 實現索引訪問和範圍檢查
- 添加迭代器支援
- 實現排序和搜索功能
- 提供統計信息（大小、容量、使用率）

## 練習 2：矩陣運算庫

實現一個矩陣運算庫：

要求：
- 支援矩陣的加法、減法、乘法
- 實現矩陣轉置、求逆
- 支援不同數據類型（int, float64）
- 實現子矩陣提取
- 添加矩陣的行列式計算
- 支援矩陣的格式化輸出

## 練習 3：環形緩衝區

實現一個環形緩衝區（循環隊列）：

要求：
- 固定大小的環形緩衝區
- 支援入隊（Push）和出隊（Pop）操作
- 實現滿了覆蓋舊數據的模式
- 支援批量讀寫操作
- 提供緩衝區狀態查詢（滿、空、使用率）
- 實現線程安全版本

## 練習 4：排序算法集合

實現多種排序算法：

要求：
- 實現冒泡排序、選擇排序、插入排序
- 實現快速排序、歸併排序、堆排序
- 支援自定義比較函數
- 添加排序性能測試
- 實現穩定排序和不穩定排序
- 支援不同數據類型的排序

## 練習 5：數據統計工具

創建數據統計分析工具：

要求：
- 計算平均值、中位數、眾數
- 實現數據分組和頻率統計
- 計算標準差和方差
- 實現百分位數計算
- 支援數據的過濾和變換
- 生成統計報告

## 練習 6：二維圖像處理

實現簡單的圖像處理功能：

要求：
- 使用二維切片表示圖像（灰度值）
- 實現圖像的旋轉、翻轉、縮放
- 實現簡單的濾波器（模糊、銳化、邊緣檢測）
- 支援圖像的裁剪和拼接
- 實現直方圖統計
- 添加圖像格式的輸入輸出

## 提示

1. 注意切片的容量管理，避免頻繁擴容
2. 正確處理邊界條件和錯誤情況
3. 理解切片和數組的本質區別
4. 善用 copy 函數避免意外的數據共享
5. 考慮內存使用和性能優化
6. 適當使用泛型提高代碼復用性
