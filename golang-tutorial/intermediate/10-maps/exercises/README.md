# 第十章練習題

## 練習 1：單詞頻率統計器

創建一個單詞頻率分析工具：

要求：
- 讀取文本內容，統計每個單詞出現的頻率
- 支援忽略大小寫和標點符號
- 實現按頻率排序功能
- 支援過濾常見停用詞（如 the, and, or 等）
- 生成頻率統計報告
- 支援多種輸出格式（文本、JSON）

## 練習 2：學生成績管理系統

實現完整的學生成績管理系統：

要求：
- 管理學生信息（姓名、學號、班級）
- 記錄多科目成績
- 計算學生平均分、總分、排名
- 統計科目平均分、最高分、最低分
- 支援按不同條件查詢（姓名、學號、班級、分數範圍）
- 生成成績報告和統計圖表

## 練習 3：LRU 緩存實現

實現一個 LRU（最近最少使用）緩存：

要求：
- 使用映射 + 雙向鏈表實現
- 支援 Get、Put、Delete 操作
- 當緩存滿時，淘汰最近最少使用的項目
- 實現緩存統計（命中率、miss 次數等）
- 支援緩存大小的動態調整
- 實現線程安全版本

## 練習 4：配置文件解析器

創建一個配置管理系統：

要求：
- 支援多種配置格式（JSON、YAML風格、ini風格）
- 實現配置的層級結構解析
- 支援環境變數覆蓋配置
- 實現配置值的類型轉換（string、int、bool、slice）
- 支援配置的驗證和默認值
- 實現熱重載功能

## 練習 5：URL 路由器

實現一個簡單的 HTTP 路由器：

要求：
- 支援靜態路由匹配
- 實現路徑參數提取（如 /user/:id）
- 支援通配符路由（如 /static/*filepath）
- 實現 HTTP 方法過濾（GET、POST等）
- 支援中間件鏈
- 提供路由組功能

## 練習 6：數據索引器

創建一個內存數據索引系統：

要求：
- 支援多字段索引建立
- 實現高效的數據查詢
- 支援範圍查詢、模糊查詢
- 實現數據的增刪改操作
- 支援復合索引
- 提供查詢性能統計

## 提示

1. 注意映射的零值特性，善加利用
2. 合理選擇鍵的類型，確保可比較性
3. 考慮併發安全，必要時使用鎖或 sync.Map
4. 注意內存使用，避免大量小對象
5. 利用映射實現集合、緩存等數據結構
6. 適當使用嵌套映射處理複雜數據
