# 第一章練習題

## 練習 1：個人介紹程序

創建一個程序，輸出您的個人信息：
- 姓名
- 年齡
- 職業
- 興趣愛好

要求：
- 使用 `fmt.Printf` 進行格式化輸出
- 使用變數存儲信息

## 練習 2：時間顯示程序

創建一個程序顯示：
- 當前日期和時間
- 今天是星期幾
- 距離新年還有多少天

## 練習 3：計算器程序

創建一個簡單的計算器：
- 定義兩個數字變數
- 計算並輸出加、減、乘、除的結果
- 使用格式化輸出顯示計算過程

## 練習 4：系統信息程序

創建一個程序顯示：
- Go 版本信息
- 作業系統信息
- 當前工作目錄
- 程序執行時的命令行參數

## 提示

1. 使用 `time` 包來處理時間相關操作
2. 使用 `os` 包來獲取系統信息
3. 使用 `runtime` 包來獲取運行時信息
4. 嘗試使用不同的 `fmt` 函數來格式化輸出

## 預期輸出範例

### 練習 1 輸出：
```
姓名：張三
年齡：25 歲
職業：軟體工程師
興趣：程式設計、閱讀、運動
```

### 練習 2 輸出：
```
當前時間：2024-01-15 14:30:25
今天是：星期一
距離新年還有：351 天
```
