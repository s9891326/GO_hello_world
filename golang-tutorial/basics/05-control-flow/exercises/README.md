# 第五章練習題

## 練習 1：猜數字遊戲

創建一個完整的猜數字遊戲：

要求：
- 程序隨機生成 1-100 之間的數字
- 用戶有最多 7 次猜測機會
- 每次猜測後給出提示（太大、太小、或猜中）
- 根據猜測次數給出不同的評價
- 詢問用戶是否再玩一局

## 練習 2：學生成績管理系統

創建一個成績管理程序：

要求：
- 輸入多個學生的多科成績
- 計算每個學生的總分和平均分
- 統計各個等級（A、B、C、D、F）的人數
- 找出最高分和最低分的學生
- 顯示班級平均分

## 練習 3：九九乘法表

創建九九乘法表程序：

要求：
- 使用嵌套循環生成九九乘法表
- 格式化輸出，對齊美觀
- 可以選擇輸出完整版或指定行
- 支援不同的顯示格式

## 練習 4：簡單計算器

創建一個菜單驅動的計算器：

要求：
- 顯示操作菜單（加、減、乘、除、退出）
- 根據用戶選擇執行相應操作
- 處理除零錯誤
- 支援連續運算
- 顯示運算歷史

## 練習 5：數字分析工具

創建一個數字分析程序：

要求：
- 判斷一個數字是否為質數
- 計算一個數字的所有因數
- 判斷是否為完全數（等於所有真因數之和）
- 判斷是否為回文數
- 計算數字的各位數字之和

## 練習 6：密碼驗證系統

創建一個密碼強度檢查程序：

要求：
- 檢查密碼長度（至少8位）
- 檢查是否包含大寫字母
- 檢查是否包含小寫字母
- 檢查是否包含數字
- 檢查是否包含特殊字符
- 根據滿足條件數量給出強度等級

## 提示

1. 使用 `math/rand` 包生成隨機數
2. 使用 `fmt.Scanf` 或 `bufio.Scanner` 讀取用戶輸入
3. 注意輸入驗證和錯誤處理
4. 善用 break 和 continue 控制循環流程
5. 使用有意義的變數名和註釋
