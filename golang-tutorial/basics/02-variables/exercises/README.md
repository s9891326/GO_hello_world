# 第二章練習題

## 練習 1：個人資料管理

創建一個程序來管理個人資料：

要求：
- 使用不同的變數聲明方式聲明以下變數：
  - 姓名（string）
  - 年齡（int）
  - 身高（float64）
  - 是否已婚（bool）
  - 電子郵件（string）
- 使用格式化輸出顯示所有信息

## 練習 2：公司配置系統

創建一個公司配置管理系統：

要求：
- 定義公司基本信息常數（公司名稱、成立年份等）
- 使用 iota 定義員工職級枚舉（實習生、初級、中級、高級、主管）
- 定義薪資等級常數
- 創建員工結構並初始化

## 練習 3：文件系統管理

創建一個文件系統管理程序：

要求：
- 使用 iota 定義文件大小單位（Byte, KB, MB, GB, TB）
- 定義文件類型常數（文檔、圖片、視頻、音頻）
- 計算並顯示不同大小文件的字節數
- 創建文件信息變數並顯示

## 練習 4：作用域實驗

創建一個程序來演示不同作用域：

要求：
- 聲明包級別變數
- 在 main 函數中聲明局部變數
- 在 if 語句塊中聲明變數
- 在 for 循環中聲明變數
- 演示變數的可見性和生命週期

## 練習 5：計算器常數

創建一個數學計算器：

要求：
- 定義數學常數（π、e、黃金比例等）
- 定義運算符常數
- 使用這些常數進行計算
- 顯示計算結果

## 提示

1. 嘗試使用所有四種變數聲明方式
2. 注意變數的命名規範
3. 理解零值的概念
4. 實驗不同作用域的變數訪問
5. 使用 iota 來創建有意義的枚舉
